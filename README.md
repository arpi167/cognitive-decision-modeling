# Cognitive Decision Modeling using Machine Learning

## Overview
This project focuses on analyzing human decision-making behavior using machine learning techniques. The system classifies individuals into different thinking styles based on their responses to scenario-based questions.

## Features
- Behavioral data analysis
- Feature engineering based on cognitive patterns
- Decision Tree classification model
- Web interface using Flask
- Insight generation based on prediction

## Dataset
The dataset consists of structured responses representing decision-making behavior across multiple scenarios.

## Methodology
1. Data preprocessing and encoding
2. Behavioral feature engineering
3. Model training using Decision Tree Classifier
4. Model evaluation and validation
5. Deployment using Flask

## Output
- Decision-making style classification
- Behavioral insight generation
- Consistency analysis

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Flask
- HTML/CSS
